  • .left300px_full_right_end { padding-left: 300px; width: 100%; -moz-box-sizing: border-box; box-sizing: border-box; background: wheat; } Why are "-moz-box-sizing: border-box; & box-sizing: border-box;" needed? The default box-model uses box-sizing-content-box which means that padding will increase the element outside its width setting. With box-sizing:border-box the padding is an inner padding and the width of the element remains while padding is changing.

  • Other ways: .content { width: 98%; /* fallback for browsers without support for calc() */ width: calc(100% - 1em); } .content { width: 100%; -moz-box-sizing: border-box; box-sizing: border-box; padding-right: 500px; margin-right: -500px; }
